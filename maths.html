Functions

Key Idea: A function maps x-values to y-values.

Example: If f(x) = 2x − 3, find f(4).

Solution: f(4) = 2(4) − 3 = 5

Common Mistake: Forgetting to substitute correctly.

Practice: Find f(−1) if f(x) = x² + 2.

Trigonometry

Key Formula: sin²θ + cos²θ = 1

Example: If sinθ = 3/5, find cosθ.

Solution: cosθ = 4/5

Common Mistake: Ignoring the quadrant.

Practice: Find tanθ if sinθ = 5/13.

Calculus

Key Rule: If f(x) = axⁿ, then f′(x) = anxⁿ⁻¹

Example: Find f′(x) if f(x) = 3x².

Solution: f′(x) = 6x

Common Mistake: Forgetting to apply the power rule.

Practice: Differentiate f(x) = 5x³.

Probability

Key Formula: P(A) = Number of favourable outcomes / Total outcomes

Example: Probability of rolling a 2 on a die.

Solution: 1/6

Common Mistake: Counting outcomes incorrectly.

Practice: Find P(heads) when tossing a coin.
