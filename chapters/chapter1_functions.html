Introduction

Functions describe how one quantity depends on another. In Grade 12, learners must understand how to analyse, interpret, and sketch different types of functions. This chapter builds understanding step by step.

Types of Functions

  • Linear functions: y = mx + c
  • Quadratic functions: y = ax² + bx + c
  • Hyperbolic functions: y = a/x
  • Exponential functions: y = abˣ
  • Inverse functions

Domain and Range

Domain is the set of all possible x-values. Range is the set of all possible y-values.

Worked Example 1

Find the domain of: f(x) = 1 / (x − 3)

Step 1: The denominator may not be zero.

Step 2: x − 3 ≠ 0

Step 3: x ≠ 3

Answer: Domain: all real numbers except x = 3

Intercepts

Worked Example 2: Finding Intercepts

Given: f(x) = 2x − 4

Y-intercept:

Step 1: Set x = 0

y = 2(0) − 4 = −4

Y-intercept: (0 ; −4)

X-intercept:

Step 1: Set y = 0

0 = 2x − 4

2x = 4

x = 2

X-intercept: (2 ; 0)

Sketching Graphs

Worked Example 3: Quadratic Function

Sketch: y = x² − 4

Step 1: Find y-intercept

x = 0 → y = −4

Step 2: Find x-intercepts

0 = x² − 4

x² = 4

x = ±2

Step 3: Determine shape

Coefficient of x² is positive → graph opens upward

Inverse Functions

An inverse function swaps x and y.

Worked Example 4

Find the inverse of: f(x) = 2x + 1

Step 1: Replace f(x) with y

y = 2x + 1

Step 2: Swap x and y

x = 2y + 1

Step 3: Solve for y

2y = x − 1

y = (x − 1)/2

Inverse: f⁻¹(x) = (x − 1)/2

Practice Exercises

  1. Find the domain of f(x) = 3/(x + 2)
  2. Determine the intercepts of y = −x + 5
  3. Sketch y = x² − 1
  4. Find the inverse of f(x) = 3x − 7

Summary

  • Functions model relationships
  • Always check domain restrictions
  • Intercepts help sketch graphs
  • Inverse functions reverse mappings
