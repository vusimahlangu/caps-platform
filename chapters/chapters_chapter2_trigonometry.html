Introduction

Trigonometry deals with the relationships between angles and sides of triangles. In Grade 12, it focuses on circular functions, identities, equations, and applications.

Circular Functions

  • Sine (sin), Cosine (cos), Tangent (tan)
  • Cosecant (csc), Secant (sec), Cotangent (cot)
  • Definitions using right-angled triangles
  • Extension to the unit circle for all angles

Trigonometric Identities

  • sin²θ + cos²θ = 1
  • 1 + tan²θ = sec²θ
  • 1 + cot²θ = csc²θ
  • Angle sum and difference formulas
  • Double angle formulas

Graphs of Trig Functions

  • Graphing sin, cos, and tan
  • Amplitude, period, phase shift
  • Identifying key points and intercepts

Solving Trig Equations

  • Using identities to simplify equations
  • Solving for general solutions
  • Applications in real-world problems

Worked Examples

Example 1: Solve sin θ = 1/2 for 0° ≤ θ ≤ 360°

Solution: θ = 30°, 150°

Example 2: Prove that 1 + tan²θ = sec²θ

Solution: Divide sin²θ + cos²θ = 1 by cos²θ → 1/cos²θ + sin²θ/cos²θ = 1/cos²θ → 1 + tan²θ = sec²θ

Practice Exercises

  1. Solve cos θ = -1/2 for 0° ≤ θ ≤ 360°
  2. Use the identity sin²θ + cos²θ = 1 to find sin²30° + cos²30°
  3. Graph y = 2sin(x) for 0 ≤ x ≤ 360°
  4. Verify the double-angle formula: sin2θ = 2sinθcosθ for θ = 45°

Summary / Key Formulas

  • sin²θ + cos²θ = 1
  • 1 + tan²θ = sec²θ
  • 1 + cot²θ = csc²θ
  • sin(A ± B) = sinA cosB ± cosA sinB
  • cos(A ± B) = cosA cosB ∓ sinA sinB
  • sin2θ = 2sinθcosθ

Chapter 2 Mini-Test

  1. Find θ if sin θ = √3/2 for 0° ≤ θ ≤ 360°
  2. Prove that 1 + cot²θ = csc²θ
  3. Solve 2cos²θ − 1 = 0 for 0° ≤ θ ≤ 360°

Answers: 1) 60°, 120°; 2) Identity verified; 3) θ = 60°, 300°
