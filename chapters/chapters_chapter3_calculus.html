Introduction

Calculus is often feared because learners are rushed through steps. This chapter slows everything down. You will see exactly where each rule comes from and how it is applied in exams.

Differentiation

Differentiation finds the rate of change or gradient of a function.

Basic Rules

  • Constant rule: d/dx(c) = 0
  • Power rule: d/dx(xⁿ) = n·xⁿ⁻¹
  • Constant multiple: d/dx(kxⁿ) = k·n·xⁿ⁻¹

Worked Example 1 (Very Slow)

Differentiate: f(x) = 3x² + 5x − 7

Step 1: Differentiate each term separately

d/dx(3x²) = 3 × 2x = 6x

d/dx(5x) = 5

d/dx(−7) = 0

Step 2: Combine results

f′(x) = 6x + 5

Applications of Differentiation

Finding the Gradient at a Point

Worked Example 2

Find the gradient of: f(x) = x² − 4x at x = 1

Step 1: Differentiate the function

f′(x) = 2x − 4

Step 2: Substitute x = 1

f′(1) = 2(1) − 4 = −2

Answer: Gradient = −2

Maximum and Minimum Values

At maximum or minimum points, the gradient is zero.

Worked Example 3

Find the turning point of: f(x) = −x² + 4x + 1

Step 1: Differentiate

f′(x) = −2x + 4

Step 2: Set derivative equal to zero

−2x + 4 = 0

−2x = −4

x = 2

Step 3: Find y-value

f(2) = −(2)² + 4(2) + 1

f(2) = −4 + 8 + 1 = 5

Turning point: (2 ; 5)

Integration

Integration is the reverse of differentiation. It is used to find areas and original functions.

Basic Rule

∫xⁿ dx = xⁿ⁺¹ / (n + 1) + C

Worked Example 4

Find: ∫(4x³ − 2x) dx

Step 1: Integrate each term

∫4x³ dx = 4 × (x⁴ / 4) = x⁴

∫−2x dx = −2 × (x² / 2) = −x²

Step 2: Add constant

Answer: x⁴ − x² + C

Area Under a Curve

Worked Example 5

Find the area under: y = x² from x = 0 to x = 2

Step 1: Integrate

∫x² dx = x³ / 3

Step 2: Substitute limits

(2³ / 3) − (0³ / 3)

= 8 / 3

Area = 8/3 square units

Practice Exercises

  1. Differentiate f(x) = 5x³ − 3x + 1
  2. Find the gradient of f(x) = x² at x = 2
  3. Find the turning point of f(x) = x² − 6x
  4. Find ∫(2x² + 4) dx

Summary

  • Differentiation finds rate of change
  • Turning points occur where gradient = 0
  • Integration reverses differentiation
  • Always include + C for indefinite integrals
