Introduction to Functions

A function is a rule that assigns to each element in a set X exactly one element in a set Y.

Functions can be represented in different ways: formula, graph, table, or mapping diagram.

Types of Functions

  • Linear Functions: f(x) = mx + c
  • Quadratic Functions: f(x) = ax² + bx + c
  • Polynomial Functions: f(x) = aₙxⁿ + … + a₀
  • Exponential Functions: f(x) = a·bˣ
  • Logarithmic Functions: f(x) = log_b(x)
  • Rational Functions: f(x) = P(x)/Q(x)

Worked Examples

Example 1: Find f(3) if f(x) = 2x + 5

Solution: f(3) = 2·3 + 5 = 11

Example 2: Determine the domain of f(x) = 1/(x-4)

Solution: x ≠ 4, so Domain: (-∞,4) ∪ (4,∞)

Practice Exercises

Summary / Key Formulas

  • Linear: f(x) = mx + c
  • Quadratic: f(x) = ax² + bx + c
  • Exponential: f(x) = a·bˣ
  • Logarithmic: f(x) = log_b(x)
  • Domain: set of x-values
  • Range: set of y-values
