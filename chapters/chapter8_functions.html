Introduction

Functions describe relationships between variables. Learners often struggle because graphs feel abstract. In this chapter, we link every equation to its graph step by step.

What is a Function?

A function assigns exactly one output value to each input value.

Notation: y = f(x)

Types of Functions (CAPS)

  • Linear: y = mx + c
  • Quadratic: y = ax² + bx + c
  • Exponential: y = abˣ
  • Hyperbolic: y = a/x

Linear Functions

Linear functions form straight-line graphs.

Worked Example 1

Given y = 2x − 4

Step 1: Identify the gradient (m)

m = 2

Step 2: Identify the y-intercept (c)

c = −4

Step 3: Draw the graph using at least two points

Quadratic Functions

Quadratic functions form parabolic graphs.

Worked Example 2

Given y = x² − 4x + 3

Step 1: Factorise

y = (x − 1)(x − 3)

Step 2: Find x-intercepts

x = 1 and x = 3

Step 3: Find y-intercept

When x = 0, y = 3

Exponential Functions

Exponential graphs show rapid growth or decay.

Worked Example 3

Given y = 2 × 3ˣ

Step 1: Identify a and b

a = 2, b = 3

Step 2: Calculate values for x = −1, 0, 1

Hyperbolic Functions

Hyperbolic functions never touch the axes.

Worked Example 4

Given y = 4/x

Step 1: Identify asymptotes

x = 0 and y = 0

Inverse Functions

Inverse functions reverse the role of x and y.

Worked Example 5

Given f(x) = 2x + 1

Step 1: Replace f(x) with y

y = 2x + 1

Step 2: Swap x and y

x = 2y + 1

Step 3: Solve for y

y = (x − 1)/2

Inverse: f⁻¹(x) = (x − 1)/2

Transformations of Functions

  • Vertical shift: y = f(x) + k
  • Horizontal shift: y = f(x − k)
  • Reflection: y = −f(x)
  • Stretch: y = af(x)

Worked Example 6

Describe the transformation from y = x² to y = x² + 3

Answer: Shifted up by 3 units

Practice Exercises

  3. Describe the transformation: y = −x²
  4. Sketch y = 2ˣ

Summary

  • Functions describe relationships
  • Graphs help visualise equations
  • Inverse functions swap x and y
  • Transformations modify graphs
